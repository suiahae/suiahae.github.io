2. 问题复现

这个错误在 conda 4.7.12 和 最新的 conda 4.8.2 中均有出现。( Windows )

使用 conda install安装其他包时, 将会出现上述问题。然而在创建一个环境时可以正常安装（例如：conda create -n tensorflow_env tensorflow）。

3. 解决办法

目前解决方案有两个，将 conda 降级、或使用 pip。

3.2 使用 pip
