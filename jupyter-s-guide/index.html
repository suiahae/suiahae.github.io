2. 卸载 jupyter

使用 pip uninstall jupyter 是卸不掉 jupyter 的，需要用到 pip-autoremove，它会卸载掉package和无用的依赖。

或者使用 conda 卸载。

3. 使用 jupyter

打开 jupyter

  • 一般可以通过在Anaconda Powershell Prompt 中输入命令 jupyter notebook。

    找到 Scripts目录下 jupyter-notebook.exe并创建快捷方式。
