为 VS Code 启用代理

VS Code 与 Google Chromium 具有完全相同的代理服务器支持。

所以

  1. VS Code 支持使用系统代理。

  2. 为 VS Code 单独设置，避免改变系统代理带来的一些问题。

  3. 需认证代理。

    使用带有用户和密码的http代理时，将弹出用户密码提示框。

    支持以下认证方法：

    • Basic
    • Digest
    • NTLM
    • Negotiate

  4. SOCKS5 不支持。
